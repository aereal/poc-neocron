# qron

qron provides [AWS CDK][aws-cdk] construct libraries that build cron-alternative consists of AWS services.

qron is aimed for the system executes tasks periodically and run in exactly-once manner.

## How to release

```
yarn
yarn build
yarn lerna publish
```
